## 🔄 COMPLETE WORKFLOW

### **For Scheduled Posts:**
```
Dashboard → Schedule Post → Database
  ↓
Scheduler runs at scheduled time
  ↓
Claude generates content + Mermaid diagram
  ↓
Diagram converted to PNG
  ↓
Approval file created with image path
  ↓
Human approves
  ↓
execute_approved.py posts with image ✅
```

## 📊 DIAGRAM KEYWORDS (Auto-Detection)

The system automatically generates diagrams when these words are detected:

```
- diagram
- flowchart
- workflow
- architecture
- process
- pipeline
- graph
- funnel
```

**Example prompts that will trigger diagram generation:**
- "Create post about cloud deployment with diagram workflow"
- "Post about our CI/CD pipeline architecture"
- "Share our sales funnel process"
- "Explain three-tier architecture with diagram"
